# 09. `for`: повторяем команду по списку

Когда в переменной лежит не одно значение, а список файлов или путей, появляется следующая задача: выполнить одну и ту же команду для каждого элемента. Здесь и нужен `for`.

В этом уроке разберём:
- `for x in $list`
- `end`
- цикл по списку файлов
- цикл вместе с уже знакомыми командами

Перейди в папку:

```bash
cd game_server_intro
```

Дальше все команды запускай внутри неё.

## Шпаргалка

- `for x in $list ... end` пройти по всем значениям в списке.

## Практика

9.1. Сохрани в переменную отсортированный список `.txt` файлов из `players/europe` и выведи каждый путь в цикле.

9.2. Сохрани в переменную список `.log` файлов и в цикле посчитай строки в каждом через `wc -l`.

9.3. Сохрани в переменную список файлов из `players/europe` и всех `.csv`, а потом в цикле посчитай байты каждого файла через `wc -c`.

9.4. Сохрани в переменную отсортированный список всех `.txt` файлов игроков и в цикле выведи для каждого строку с префиксом `profile:`.
