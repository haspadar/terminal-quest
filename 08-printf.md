# 08. `printf`: выводим по шаблону

`echo` хорош для быстрой проверки, но как только нужен аккуратный и предсказуемый вывод, лучше брать `printf`. Он позволяет сразу собрать строку в нужном виде: с подписью, путём, числом или результатом команды.

В этом уроке разберём:
- `%s`
- `\n`
- вывод переменной по шаблону
- вывод результата команды по шаблону

Перейдите в папку:

```bash
cd game_server_intro
```

Дальше все команды запускайте внутри неё.

## Шпаргалка

- `printf '%s\n' $name` вывести значение с переводом строки.
- `printf 'x: %s\n' $name` вывести подпись и значение.

## Практика

8.1. Сохраните путь `players/europe` в переменную и выведите его через `printf` с префиксом `scan:`.

8.2. Сохраните в переменную результат `wc -l chat/global_chat.log` и выведите его через `printf` с префиксом `chat stats:`.

8.3. Сохраните в переменную отсортированный список `.txt` файлов из `players/europe` и выведите каждый путь через `printf`.

8.4. Сохраните в переменную список файлов из `players/europe` и всех `.csv`, а потом выведите этот список через `printf` с одним и тем же префиксом для каждого значения.
