# 11. `for`: повторяем команду по списку

Когда в переменной лежит не одно значение, а список файлов или путей, появляется следующая задача: выполнить одну и ту же команду для каждого элемента. Здесь и нужен `for`.

В этом уроке разберём:
- `for x in $list`
- `end`
- цикл по списку файлов
- цикл вместе с уже знакомыми командами

Перейдите в папку:

```bash
cd game_server_intro
```

Дальше все команды запускайте внутри неё.

## Шпаргалка

- `for x in $list ... end` пройти по всем значениям в списке.

## Практика

11.1. Сохраните в переменную отсортированный список `.txt` файлов из `players/europe` и выведите каждый путь в цикле.

11.2. Сохраните в переменную список `.log` файлов и в цикле посчитайте строки в каждом через `wc -l`.

11.3. Сохраните в переменную список файлов из `players/europe` и всех `.csv`, а потом в цикле посчитайте байты каждого файла через `wc -c`.

11.4. Сохраните в переменную отсортированный список всех `.txt` файлов игроков и в цикле выведите для каждого строку с префиксом `profile:`.
