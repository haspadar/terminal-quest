# 13. `xargs`: передаём список дальше

До этого момента команды либо искали, либо считали, либо сортировали. Но часто задача звучит так: “нашёл список файлов, теперь примени к каждому ещё одну команду”. Здесь и появляется `xargs`.

Перейдите в папку:

```bash
cd game_server_intro
```

## Самая простая связка

```bash
find players -name '*.txt' | xargs wc -l
```

Должно быть: для каждого профиля выводится `8`, а внизу общая сумма `56`.

`find` отдаёт список файлов, а `xargs` превращает этот список в аргументы для `wc -l`.

## Передать список в `grep`

```bash
find logs -name '*.log' | xargs grep -H 'ERROR'
```

Должно быть: находятся строки с `ERROR` из дневных логов.

Это уже обычная рабочая сцепка: одна команда нашла файлы, вторая ищет внутри них.

## Передать список в `cat`

```bash
find players/europe -name '*.txt' | xargs cat
```

Должно быть: подряд выводится содержимое трёх профилей из `players/europe`.

## Шпаргалка

- `find ... | xargs wc -l` передать найденные файлы в `wc`.
- `find ... | xargs grep 'x'` передать найденные файлы в `grep`.
- `... | xargs cat` передать список в `cat`.

## Практика

1. Найдите все `.txt` профили игроков и посчитайте строки в каждом через `xargs`.

Должно быть: у каждого файла `8` строк, общая сумма `56`.

2. Найдите все `.log` файлы и покажите в них строки с `ERROR` через `xargs`.

Должно быть: совпадения находятся во всех трёх дневных логах.

3. Найдите все профили игроков из Европы и выведите их содержимое через `xargs`.

Должно быть: выводится содержимое `anna.txt`, `ivan.txt`, `max.txt`.

4. Найдите все профили игроков из Европы и посчитайте, сколько строк в них суммарно.

Должно быть: `24`.
