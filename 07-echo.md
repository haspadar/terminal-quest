# 07. `echo`: быстро показываем значения

Переменные уже есть, но теперь нужно быстро проверить, что в них лежит. Для этого и нужен `echo`: он помогает мгновенно вывести значение на экран и убедиться, что вы сохранили именно то, что хотели.

В этом уроке разберём:
- `echo $name`
- быстрый вывод строки и переменной
- проверку переменных перед следующей командой

Перейдите в папку:

```bash
cd game_server_intro
```

Дальше все команды запускайте внутри неё.

## Шпаргалка

- `echo $name` вывести значение переменной.
- `echo text $name` вывести текст и значение в одной строке.

## Практика

7.1. Сохраните путь `players/europe` в переменную и выведите его через `echo`.

7.2. Сохраните в переменную отсортированный список `.txt` файлов из `players/europe` и выведите его через `echo`.

7.3. Сохраните в переменную результат `wc -l chat/global_chat.log` и выведите его через `echo`.

7.4. Сохраните в переменную список файлов из `players/europe` и всех `.csv`, а потом выведите эту переменную через `echo`.
