# 07. `echo`: быстро показываем значения

Переменные уже есть, но теперь нужно быстро проверить, что в них лежит. Для этого и нужен `echo`: он помогает мгновенно вывести значение на экран и убедиться, что ты сохранил именно то, что хотел.

В этом уроке разберём:
- `echo $name`
- быстрый вывод строки и переменной
- проверку переменных перед следующей командой

Перейди в папку:

```bash
cd game_server_intro
```

Дальше все команды запускай внутри неё.

## Шпаргалка

- `echo $name` вывести значение переменной.
- `echo text $name` вывести текст и значение в одной строке.

## Практика

7.1. Сохрани путь `players/europe` в переменную и выведи его через `echo`.

7.2. Сохрани в переменную отсортированный список `.txt` файлов из `players/europe` и выведи его через `echo`.

7.3. Сохрани в переменную результат `wc -l chat/global_chat.log` и выведи его через `echo`.

7.4. Сохрани в переменную список файлов из `players/europe` и всех `.csv`, а потом выведи эту переменную через `echo`.
