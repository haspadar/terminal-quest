# 15. `z`: умная навигация по папкам

`cd` требует полного пути или многих нажатий Tab. `z` запоминает папки, в которых вы уже бывали, и позволяет переходить к ним по короткому фрагменту имени.

## Установка

В fish — через fisher:

```bash
fisher install jethrokuan/z
```

На других оболочках — через пакетный менеджер:

```bash
brew install zoxide   # zoxide — более современный аналог
```

## Как работает

`z` накапливает историю переходов через `cd` и ранжирует папки по «frecency» — алгоритму, который учитывает и частоту визитов, и их давность. Чем чаще и недавнее вы были в папке — тем выше она в списке.

## Базовое использование

Сначала побывайте в папке через обычный `cd`:

```bash
cd /путь/до/terminal-quest/game_server_intro/logs
cd /путь/до/terminal-quest/game_server_intro
```

Теперь `z` знает об этой папке. Перейдите в неё по фрагменту:

```bash
z logs
```

Должны оказаться внутри `game_server_intro/logs` без ввода полного пути.

## Показать совпадения

```bash
z -l logs
```

Должны быть видны все папки с `logs` в имени из истории, отсортированные по frecency.

Полезно, когда несколько папок подходят под фрагмент и хочется выбрать нужную.

## Шпаргалка

- `z fragment` — перейти в папку по фрагменту имени.
- `z -l fragment` — показать совпадения без перехода.

## Напоследок

- `z` не заменяет `cd` — она дополняет: `cd` строит историю, `z` по ней перемещается.
- Если под фрагмент подходит несколько папок, `z` выбирает ту, у которой выше frecency. `z -l` покажет всех кандидатов.

## Практика

15.1. Перейдите в `game_server_intro/logs` через полный `cd`, затем вернитесь выше и снова попадите туда через `z`.

   `z logs` должна привести вас в `logs` без полного пути.

15.2. Перейдите в несколько разных папок датасета через `cd`, затем выведите список кандидатов для одного из фрагментов через `z -l`.

   Должны быть видны папки с frecency-счётом рядом с путём.

15.3. Попробуйте перейти через `z` в папку, в которой ещё не бывали через `cd`.

   `z` должна сообщить, что не нашла совпадений — она работает только по истории.
