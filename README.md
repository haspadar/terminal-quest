# Terminal Quest

Курс по командной строке.

Научимся анализировать данные на практике:
- понимать структуру директорий,
- находить файлы,
- искать по содержимому,
- считать найденное,
- сортировать,
- собирать пайплайны

Курс рассчитан на тех, кто уже знаком с основами:
- что такое shell (запуск команд, аргументы, флаги)
- как перемещаться по файловой системе (`pwd`, `cd`, `ls`)
- как читать и редактировать файлы (`cat`, `less`, `nano` или `vim`)
- потоки и пайпы (`stdin`, `stdout`, `stderr`, `|`)
- базовый поиск по тексту (`grep`, простые шаблоны).
- регулярные выражения.

Примеры рассчитаны на Ubuntu, WSL Ubuntu и Arch Linux.
Рабочая оболочка — `fish`. Большинство команд из уроков будут работать и в других shell, но примеры записи переменных, циклов и подстановок даны для `fish`.

Для первых уроков — архив `game_server_intro.zip`.

## OpenSearch

OpenSearch — AI-помощник для терминала и кода.
В этом курсе OpenSearch объяснит ошибку, подскажет следующий шаг или проверит ход мысли.

1. Откройте новую вкладку терминала.
2. Перейдите в папку репозитория:

```bash
cd terminal-quest
```

3. Запустите OpenSearch:

```bash
opensearch
```

После этого откроется агент. Файл урока держите рядом в редакторе или отдельной вкладке.

В этом репозитории агент сам отвечает как учебный помощник. Достаточно написать номер задачи, свой вариант команды или короткий вопрос. Ответ должен идти по шагам, с объяснением и без готового решения по умолчанию.

Пример:

```text
задача 1.4
моя команда: tree logs -L 2
что не так?
```

Ответ:

```text
Вы смотрите только внутрь `logs`.
Здесь задача про весь набор данных.
Проверьте, нужен ли вам тут аргумент `logs`.
```
