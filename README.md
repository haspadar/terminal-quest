# Terminal Quest

Курс по командной строке.

Научимся анализировать данные на практике:
- понимать структуру директорий, 
- находить файлы, 
- искать по содержимому, 
- считать найденное, 
- сортировать,
- собирать пайплайны

Курс рассчитан на тех, кто уже знаком с основами:
- что такое shell (запуск команд, аргументы, флаги)
- как перемещаться по файловой системе (`pwd`, `cd`, `ls`)
- как читать и редактировать файлы (`cat`, `less`, `nano` или `vim`)
- потоки и пайпы (`stdin`, `stdout`, `stderr`, `|`)
- базовый поиск по тексту (`grep`, простые шаблоны).
- регулярные выражения.

Примеры работают на Linux и macOS.
Рабочая оболочка — `fish`. Впрочем, подойдёт и любой другой shell.

Для первых уроков — архив `game_server_intro.zip`.

## Как скачать

Установите инструменты:

```bash
sudo apt update
sudo apt install fish git nodejs npm
```

Затем скачайте репозиторий:

```bash
git clone https://github.com/haspadar/terminal-quest.git
cd terminal-quest
```

## Kilo

Kilo — AI-помощник для терминала и кода.
В этом курсе Kilo — собеседник рядом: объяснит ошибку, намекнет следующий шаг или проверит ход мысли.

Запустите `kilo` в одной вкладке, а урок — в другой.

1. Установите Kilo CLI:

```bash
sudo npm i -g @kilocode/cli
```

2. Перейдите в директорию репозитория:

```bash
cd terminal-quest
```

3. Запустите Kilo:

```bash
kilo
```
Выберите настройки по умолчанию.

В репозитории есть AI-ментор. К нему обращаются по задачам и командам по имени `mentor` или `ментор`. Он подсказывает постепенно и не даёт готовое решение.

Пример:

```text
mentor, задача 1.4
моя команда: tree logs -L 2
что не так?
```

Ответ:

```text
Вы смотрите только внутрь `logs`.
Здесь задача про весь набор данных.
Проверьте, нужен ли вам тут аргумент `logs`.
```
